# Setting profile 
- Temuan Bug
    - Edit password muncul error ( Tidak muncul dialog Sukses dan tidak hapus session Login/redirect Halaman login )
    - Batal Edit Password Muncul Error
    - Edit Nama Muncul Error ( Berhasil Simpan Namun tidak muncul dialog sukses )
    - Batal Edit Nama Muncul Error 

# Metode Pembayaran Offline
- sukses tanpa bug
    - Tambah Data berhasil dan muncul
    - Hapus data Normal
- Temuan Bug
    - Ubah data Berhasil namun tidak muncul perubahan

# Metode Pembayaran Ketika Online
- Sukses Tanpa Bug
    - Tambah data
    - Ubah Data
    - Delete data
- Temuan Bug

# Titipan Offline
- sukses tanpa bug
    - Tambah Titipan Berhasil
        - Perhitungan masuk/keluar atau tambah pergerakan pada offline berjalan
# Titipan Online
- Temuan Bug saat Online
    - Perhitungan langsung menjadi 0
    - Log Masih tersimpan namun data total pergerakan tidak dihitung

# Transaksi Offline
- Sukses bug 
 - Tambah Ubah transaksi berjalan tanpa error
# Transaksi Online
- Sukses tanpa bug
    - Semua histori transaksi tidak ada perubahan dan masih sama
